Table of endpoints with their request methods

Request URL HTTP Method Query Parameters Request Data Format Response Data Body Format
Return a list of ALL movies /movies GET None None A JSON object holding data about all the movies
Return data of a single movie by title /movies/:title GET :title None A JSON object holding data about a single movie
Return data of genre by title /movies/:title/genre GET :title None A JSON object holding data of genre about a single movie
Allow new users to register /users POST none JSON object holding data about the user structured like: {"name": "Peter Parker"} A JSON object holding data of the new user including id
Allow users to update their user info /users/:name PUT :name JSON object holding data about the requested new username structured like: {"newName": "Peter Parker"} A message showing the old and new username
Allow users to add a movie to their list of favorites /users/:name/add POST :name JSON object holding data about the movie to be added structured like: {"movie": "Spider-man"} A message showing the added movie
Allow users to remove a movie from their list of favorites /users/:name/remove DELETE :name JSON object holding data about the movie to be added structured like: {"movie": "Spider-man"} A message showing the removed movie
Allow existing users to deregister /users/:name/deregister DELETE :name None A message showing the name that has been deleted
